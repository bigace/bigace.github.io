This video demonstrates how to install BIGACE on your webhost. You see how and where to download the latest BIGACE version, how to extract the files, upload them using FTP and then the browser-based installation process.

Screencast: Installation of BIGACE

Don't like the quality? If you want to watch the video in HD, please use the link to go to BIGACE Web CMS on Vimeo directly or use this link to get to the Screencast: BIGACE Web CMS - Installation. You can also download the HD version of the video, use the link on the video page.
