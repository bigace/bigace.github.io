As development with Bigace 3.0 goes on, the improvement of the 2.7 version slows down.

But there is another version out now, which fixes a couple of Bugs and introduces new languages in (pre-)installer and as extension.

Thanks to the High-Tech Bridge SA from Geneva, Switzerland we could fix a path disclosure problem in some scripts. Please note that it was only a problem for installations where .htaccess security was not activated during installation!

Some of the new features:

  • included new translations in pre-installer (now available: German, English, Spanish, Portuguese, Swedish, Finnish, Russian)
  • included new translations in installer (now available: German, English, Spanish, Russian, si, hr, Italian, Finnish, Romanian, Turkish)
  • using path constants instead of global variables (aka code cleanup)
  • broken editor in some rare translation circumstances
  • and some other small bugfixes - but I forgot to write them down...

Okay, now go and grab your download :D
