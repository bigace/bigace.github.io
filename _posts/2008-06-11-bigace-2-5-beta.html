20 days after the alpha version, here is the official beta version.

Depending on the amount of feedback this could be the latest version before the official release ... so please everyone:

*** Install it at least once and run some basic tests ***

Improvements

This is a short outtake of the most important changes since the alpha version:

  • Multiple contents for each page can be customized based on Smarty designs (edit them in the Administration panel)
  • New pluggable Captcha System, supporting the new Securimage Extension, please test this as well!
  • A new Filemanager release is there, please test it too.
  • Tons of translation fixes
  • Easified installer, checking for correct PHP version >= 5.1
  • {sitename} TAG, which is supported through installation
