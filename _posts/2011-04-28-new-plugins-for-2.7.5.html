After the last release of several plugins updates and Bigace 2.7.5, its again time to share the latest versions with you!

Here are the new extensions:

  • YouTube 1.1 - a plugin that lets you easily include YouTube videos
  • Audio Player 1.3 - portlet and content parser that supports MP3 playback
  • Lotusflower 1.1 - a smarty template by Oleg (originally made for www.ribalka-a.ru)
  • Music-Website 1.1 - a smarty template for small music websites
  • Bike Riders 1.0 - a smarty template for biking websites (including drop-down menus)

And here are the updated extensions:

  • News 1.6 - an updated version for Bigace 2.7.5
  • Social-Bookmarks 0.5 - inserts buttons for Social Bookmarks to your pages
  • Submenu Preview 1.3 - a module to list all pages below the current one
  • CrystalX 1.4 - improved version by Dragonslayer (see Forum)
  • Filemanager 0.9 - a replacement for the default editor link and image dialogs
  • Photo Album 1.2 - a simple HTML based and CSS styleable foto gallery
  • Russian 1.3 - russian translations

Have fun! And please leave your feedback in the forum if you find any problem.
